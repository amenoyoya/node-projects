# Project generators for Node.js

## What's this?

Node.jsの開発環境でよく使うプロジェクトの最小構成を生成するスクリプト集

***

## Environment

- CLI:
    - nodejs: `10.15.3`

***

## Project templates

### Vuetify
```bash
$ node vuetify.js

# => generate minimal vuetify project
# ./
# |- public/
# |   `- index.html
# |- src/
# |   |- App.vue
# |   `- index.js
# |- package.json
# `- webpack.config.js
```
